## Product / AI Product

Core formula: problem discovery + requirement definition + cross-functional execution + measurable product outcome.

Evidence markers:

- user research, feedback synthesis, persona or scenario definition,
- PRD, roadmap, feature priority, acceptance criteria,
- engineering/design/data coordination,
- launch, experiment, activation, retention, conversion, cost, quality metrics.

Preferred verbs: defined, prioritized, validated, launched, iterated, 推动, 拆解, 设计, 落地, 复盘.

Common mistakes:

- writing only "wrote PRD" without problem or outcome,
- listing AI tools without product judgment,
- claiming ownership without cross-functional evidence.

## Operations / Growth

Core formula: user or supply problem + operating mechanism + data feedback + growth/efficiency result.

Evidence markers:

- campaign design, channel operation, community/user lifecycle,
- SOP, content calendar, segmentation, funnel, conversion,
- cohort analysis, retention, activation, GMV, leads, cost per acquisition.

Preferred verbs: segmented, executed, optimized, scaled, retained, 运营, 触达, 转化, 拉新, 留存, 搭建机制.

Common mistakes:

- activity description without metric,
- vanity metrics without business meaning,
- no segmentation logic.

## Data Analyst

Core formula: business question + data extraction/cleaning + analysis method + decision impact.

Evidence markers:

- SQL, Python, dashboard, ETL, BI tools,
- metric definition, experiment analysis, funnel/cohort/retention,
- anomaly diagnosis, stakeholder recommendation, decision support.

Preferred verbs: queried, modeled, diagnosed, visualized, automated, 分析, 建模, 搭建看板, 定义指标.

Common mistakes:

- listing tools without analysis use case,
- charts with no decision,
- percentages without denominator.

## Software / AI Engineer

Core formula: technical problem + design/implementation + reliability/performance + shipped impact.

Evidence markers:

- architecture, API, model pipeline, evaluation, deployment,
- latency, accuracy, cost, uptime, test coverage, scalability,
- code ownership, debugging, monitoring, documentation.

Preferred verbs: implemented, optimized, refactored, deployed, integrated, built, 设计, 实现, 优化, 部署.

Common mistakes:

- project stack dump without engineering problem,
- no measurable performance or reliability outcome,
- claiming "full stack" without shipped features.

## Marketing / Brand

Core formula: audience insight + message/creative strategy + channel execution + measurable reach or conversion.

Evidence markers:

- positioning, campaign, content, social, KOL, SEO/SEM,
- audience segmentation, creative testing, traffic, leads, engagement,
- brand consistency, launch plan, post-campaign analysis.

Preferred verbs: positioned, planned, produced, amplified, converted, 策划, 投放, 传播, 复盘.

Common mistakes:

- only listing content produced,
- no audience or channel logic,
- no outcome beyond views.

## Design / UX

Core formula: user problem + research/interaction solution + prototype/spec + usability or business result.

Evidence markers:

- interviews, journey map, wireframe, prototype, design system,
- Figma, usability testing, accessibility, handoff,
- task completion, conversion, satisfaction, defect reduction.

Preferred verbs: researched, prototyped, tested, redesigned, standardized, 调研, 原型, 交互, 走查.

Common mistakes:

- visual polish without user problem,
- no design rationale,
- no collaboration with product/engineering.

## Consulting / Finance / Research

Core formula: question/hypothesis + structured analysis + evidence synthesis + recommendation or investment/business implication.

Evidence markers:

- market sizing, comparable analysis, financial modeling, due diligence,
- interviews, desk research, policy/industry synthesis,
- memo, deck, recommendation, risk assessment.

Preferred verbs: assessed, modeled, synthesized, recommended, evaluated, 测算, 拆解, 研判, 撰写.

Common mistakes:

- generic research wording,
- no methodology,
- no implication or recommendation.
